w => Bir sonraki kelimeye git
W => Boşlukla ayrılmış olan bir sonraki kelimeye git
e => kelimenin sonuna git
0 => satır başı
$ => satır sonu
+ => Bir sonraki satır başı
- => Bir önceki satır başı
H => Ekranın en üstündeki satıra git
M => Ekranın ortasındaki satıra git
L => Ekranın en altındaki satıra git
:n => n. satıra git
[C-d] => yarım ekran aşağı kaydır
[C-u] => yarım ekran yukarı kaydır
zz => imleci ortala
v => visual mode (seçim ilemleri burda yapılıyor)
/n => n i ara
n => aramadaki bi sonraki kelime
N => aramadaki bi önceki kelime
i => imlecin olduğu yere yaz
I => satır başına yaz
a => imlecin bi sağına yaz
A => satır sonuna yaz
o => bir alttaki satıra yaz
O => bir üstteki satıra yaz
r => karakteri değiştir
R => karakterlerin üzerine yaz(replace)
cw => imlecin sağındaki kelimeyi sil ve yazmaya başla
cc => satırı sil ve yazmaya başla
C => imlecin sağında kalan herşeyi sil ve yazmaya başla
x => Sağa doğru bir karakter sila
X => Sola doğru bir karakter sil
dw => Kelime sonuna kadar sil
dd => Tüm satiri sil
D => imleçten satır sonuna kadar sil
yw => sonraki kelimeyi kopyala
yy => Tüm satırı koplala
p => Silinen satırı alt satıra yapıştır
P => Silinen satırı üst satıra yapıştır
U => işlemi geri al
[C-r] => geri alınan işlemi geri al
